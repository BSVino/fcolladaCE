00053 /* 
00054  * The following macro invocations define the supported CG basic data types.
00055  *
00056  * The macros have the form :
00057  *
00058  *   CG_DATATYPE_MACRO(name, compiler_name, enum_name, base_name, nrows, ncols, pc_name)
00059  *
00060  *     name          : The name of the data type.
00061  *     compiler_name : The name of the data type within the compiler syntax.
00062  *     enum_name     : The C enumerant.
00063  *     base_name     : The C enumerant of the base type.
00064  *     nrows         : Number of rows for matrix types.  Should be 0 other-wise.
00065  *     ncols         : Number of columns for scalar, vector, and matrix types.
00066  *     pc_name       : The C enumerant of the parameter class.
00067  *   
00068  */
